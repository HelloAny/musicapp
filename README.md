1. Clone this repo

```bash
git clone https://github.com/zhuyunzhang/musicapp.git
```

2. Change into the directory

```bash
cd "目标目录下"
```

3 Install the dependencies

```bash
yarn
# or
npm install
```

4. Run the project

```bash
react-native run-ios
# or
react-native run-android
```
